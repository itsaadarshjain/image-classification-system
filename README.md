# Image Classification System

A Deep Learning project that classifies images from the CIFAR-10 dataset using a Convolutional Neural Network (CNN) built with PyTorch.

## Project Overview
This project focuses on image classification using CNN. The model learns visual features like edges, patterns, and shapes to classify images into different categories.

## Dataset
Dataset: CIFAR-10  
Classes: airplane, automobile, bird, cat, deer, dog, frog, horse, ship, truck

## Technologies Used
- Python
- PyTorch
- Torchvision
- NumPy
- Matplotlib
- Jupyter Notebook

## Key Features
- CNN model implementation
- Data preprocessing
- Data augmentation
- Model training and validation
- Accuracy and loss evaluation
- Best model saving using `best_model.pt`

## Model Architecture
- Convolution layers
- ReLU activation
- MaxPooling
- Fully connected layers
- Output layer for 10 classes

## Results
Accuracy: 78.44%
